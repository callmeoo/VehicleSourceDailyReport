# 车源部门业务日报表

本目录用于沉淀「经管报表-车源日报表」需求说明、附件模板和前端静态原型。

## 文件说明

- `车源部门业务日报表需求说明.md`：当前最新需求口径，包含查询条件、报表结构、指标口径、权限、初始化、导出和原型评审说明。
- `车源日报表原型.html`：前端静态原型，直接用浏览器打开即可预览。
- `车源部门业务日报表.xlsx`：原始附件模板。

## 当前确认口径

- 功能入口：在 BMS 系统 `经管报表` 页面，于 `车源上拍明细` Tab 前新增 `车源日报表` Tab。
- 时间范围：限制为单月，表头固定展示 `合计` 和 `1` 至 `31` 日。
- 时间交互：使用年月选择器，只能选择单个月份，不支持跨月范围。
- 第一列：展示 `业务指标`。
- 行指标：检测数量、上拍数量、成交数量、总毛利、台均总毛利、成交率。
- 合计口径：台均总毛利、成交率直接沿用成绩单合计口径，不累加每日值。
- 权限：Tab 查看权限、按大区域/小区域/门店的数据权限、导出独立权限。
- 初始化：功能上线时初始化 `2025年1月` 至上线前月份的数据。

## 原型交互

- 大区域、小区域、拍卖方式使用 mock 下拉数据；小区域随大区域联动刷新。
- 所属集团、门店全称、总店为输入框。
- 表格标题旁展示所选月份，例如 `所选月份：2026年-5月`。
- 页面展示业务指标趋势折线图，横轴为日期，左侧主坐标为台次，右侧副坐标为成交率，每隔 5 天标注当日数值，并支持点击指标显示或隐藏对应折线。
- `需求点说明` 使用弹框展示，入口和弹框内需求点标题均为红色加粗。

## 评审入口

打开 `车源日报表原型.html`，重点评审：

- 查询条件字段和控件类型是否符合业务习惯。
- 年月选择器是否满足单月查询场景。
- 报表矩阵的表头、行指标和合计口径是否清晰。
- 业务指标趋势图是否有助于观察每日变化。
- 权限、导出和初始化要求是否覆盖研发实现范围。
